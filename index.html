Canal

"CANAL Cobol Analyzer" is a tool for editing and analyzing COBOL files

  • Incredibly fast loading and syntax highlighting thanks to FastColoredTextBox
  • Beautiful code thanks to Source Code Pro font
  • Resolving COPYs with just one click
  • Extensive automatic analysis, allowing you to
    • View varibale definition on double click
    • See perform tree (which procedures call which other procedures?)
    • Find all variables used in a procedure, and if they are input or output
  • Including code generator to create C#-enums from level 88 structures

Installation via www.jannikarndt.de/media/canal/Canal.application, requires .NET 4.5
