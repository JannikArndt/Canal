Canal

CANAL Cobol Analyzer is an open source tool for editing and analyzing COBOL files, written by someone, who does that every day.

  • Incredibly fast loading and syntax highlighting thanks to FastColoredTextBox.
  • Beautiful code thanks to Source Code Pro font and automatic formatting.
  • Easily insert Copybooks into main code or open them in another tab.
  • Extensive automatic analysis, allowing you to
    • View variable definition on double click (and jump to definition –even in copybooks– with just another click),
    • See perform tree (which procedures call which other procedures?),
    • Find all variables used in a procedure, and if they are input or output.

Installation via www.jannikarndt.de/media/canal/Canal.application, requires .NET 4.5
